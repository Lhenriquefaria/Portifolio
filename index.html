Criando sua carreira

Desenvolvi um mini jogo interativo utilizando HTML (index.html), CSS (style.css) e JavaScript (script.js). O jogo simula uma final do Paulistão entre Corinthians e Palmeiras, onde o jogador toma decisões em momentos cruciais, como escolher o time, o lado do pênalti ou onde bater a falta. Cada escolha influencia o resultado, determinando se o jogador será campeão ou não. O objetivo foi criar uma experiência envolvente e dinâmica, usando lógica de programação e interação com o usuário.

Timãonews

Desenvolvi um site simples utilizando HTML (index.html) e CSS (style.css), com o objetivo de reunir as últimas notícias do Corinthians por meio de vídeos do YouTube. A página exibia uma seleção de vídeos atualizados, e ao clicar em um deles, o usuário era redirecionado diretamente para o YouTube. Foi uma forma prática de centralizar conteúdos recentes sobre o time, com foco em organização visual e navegação fácil.
